# 🚀 HireHub AI

An AI-powered Full Stack MERN Job Portal built for modern recruitment.

HireHub AI helps candidates build professional profiles, upload resumes, analyze ATS scores using AI, and apply for jobs while enabling recruiters to manage hiring efficiently.

---

# 📌 Features

## Candidate

- User Authentication (JWT)
- Candidate Profile
- Resume Upload
- AI Resume Analysis
- ATS Score
- Job Search
- Apply for Jobs
- Application Tracking
- Dashboard

---

## Recruiter

- Recruiter Authentication
- Company Profile
- Post Jobs
- Edit Jobs
- Delete Jobs
- Manage Applicants
- AI Resume Ranking
- Dashboard

---

## Admin

- Manage Users
- Manage Jobs
- Platform Analytics

---

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/HireHub-AI.git
```

---

## Backend

```bash
cd backend
npm install
npm run dev
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file.

```
PORT=

MONGODB_URI=

JWT_ACCESS_SECRET=

JWT_REFRESH_SECRET=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=

GEMINI_API_KEY=
```
